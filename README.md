# Week2 : Linked-List
## **About Linked-List**

* Linked list 是一種常見的資料結構，使用node(節點)來記錄、表示及儲存資料，並利用每個node中的pointer(指標)指向下一個node，藉此將多個node串連起來，形成Linked list，並以NULL來代表Linked list的終點。
